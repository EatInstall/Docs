Get Eat

NOTE: Eat is in alpha stage. We cannot be liable for any bugs in the package manager. Use Eat at your own risk.

Eat is supported on hundreds of Linux distributions. Originally, it only worked on Debian and derivatives of Debian (e.g. Ubuntu).

Eat has a simple installer maintained by our community. It depends on dialog. To install dialog, use APT: sudo apt install dialog

Learn more about the APT CLI

Then to install Eat, run the installer:

bash -c "$(curl -fsSL https://raw.githubusercontent.com/EatInstall/EatInstaller/HEAD/eatinstaller.sh)"
ERRORS

"Distribution Not Supported": Eat has no support for your distribution. This error usally appears on Gentoo Linux, we are working to see if we can support Gentoo in the future.

git: command not found: It's probably a bug in your package manager. If your distro is Debian or Ubuntu, try sudo apt reinstall git and Git should be working.

Warning: Update check failure: This is possibly when new files are added to Eat's source (e.g. adding a new package management utility) or when files are removed from the source distribution. Please cd into ~/Eat-PKG-Manager and run git pull.
